# Bookstore Database Project

A simple Microsoft Access database project for managing bookstore records. It stores books, authors, and publishers in 
related tables and uses queries to organize and find information easily.

I originally built this project in high school as part of learning basic database design and data management.

## What It Does

- Stores book information in a structured database
- Organizes authors, publishers, and books into related tables
- Uses queries to filter and view specific groups of books
- Demonstrates basic relational database design in Microsoft Access

## Main Features

- `Author` table
- `Publisher` table
- `Bookstore` table
- Saved queries for browsing and filtering records

## Tools Used

- Microsoft Access
- Relational tables
- Queries
- Basic database relationships

## Why I Made It

I built this project to practice how databases work beyond spreadsheets. 
It helped me learn how to organize information, connect related records, and retrieve data more efficiently.
